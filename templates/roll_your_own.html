Roll Your Own

You don't need this page. {{ brand }} writes the filter file for you — just download and import. But if you're the type who wants to see how the sausage is made (or hand-edit it), here's how Gmail filters actually work.

1. The file is XML

Gmail imports and exports filters as an XML file. You never have to read it, but here's what one filter looks like inside:

<entry>
  <category term='filter'/>
  <apps:property name='from'  value='chase.com'/>
  <apps:property name='label' value='Banking'/>
</entry>

To open or tweak it by hand, any text/XML editor works — VS Code (Mac/Windows/Linux), Notepad++ (Windows), or paste it into a free in-browser viewer like CodeBeautify — no install.

2. Matching several senders in one rule (the { } trick)

To file mail from many senders to one label, you might reach for OR. Don't — in Gmail, OR mixed with parentheses leaks out of the field and starts searching the whole email body, which causes wild false positives:

from: ("chase.com") OR ("citi.com")   ← leaks: searches the body for "citi.com"
from: {"chase.com" "citi.com"}        ← airtight: bound to the From field

The brace syntax binds the OR strictly to that field. {{ brand }} generates the brace form for you when you consolidate senders — so you never have to remember this.

3. Gmail caps you at ~500 filters

Gmail allows roughly 500 filters per account (and a separate limit of 20 filters that forward mail). If you're a heavy filterer, that ceiling is real. Google's own advice when you approach it: consolidate with the { } grouping above, and filter by domain (chase.com) rather than individual addresses. One brace rule can hold dozens of senders and counts as a single filter.

Works even with Advanced Protection. Google's Advanced Protection Program blocks apps — even ones you wrote — from touching your mailbox. So every "connect your Gmail" tool simply doesn't work for those accounts. {{ brand }} never asks for access to your account: filters are built in your browser and you import the file yourself. Nothing to authorize, nothing that can break.
