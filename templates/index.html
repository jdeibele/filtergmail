How to import into Gmail

  1. Click Download Gmail filters XML above — save the file anywhere.
  2. In Gmail, click the gear icon ⚙ → See all settings
  3. Click the Filters and Blocked Addresses tab
  4. Scroll to the bottom → click Import filters
  5. Choose the downloaded file → Open file → Create filters
What's in the file? (it's XML)

Your filters are saved as XML — Gmail's import format. You never need to open or edit it; just download above and import. For the curious, one filter looks like this inside:

<entry>
  <category term='filter'/>
  <apps:property name='from'  value='chase.com'/>
  <apps:property name='label' value='Banking'/>
</entry>

Want to hand-tweak it? Open it in any text/XML editor — e.g. VS Code (Mac/Windows/Linux), Notepad++ (Windows), or paste it into a free in-browser viewer like CodeBeautify — no install.
